# Nightly Countdown

A Firefox extension that shows how many days remain until the next Firefox Nightly release cycle starts.

## What it does

- Displays a toolbar icon with a **colored arc** and a **day count** at its centre:
  - The arc spans from the top of the circle and shrinks as the next Nightly date approaches.
  - The color transitions from green (plenty of time) to red (imminent).
  - A grey ring is shown when data cannot be fetched.
- Clicking the icon opens a popup with:
  - The next Firefox version number
  - The date Nightly starts
  - The number of days remaining
- Data is fetched from [whattrainisitnow.com](https://whattrainisitnow.com) and refreshed once per day.

## Project structure

```
webextension/
├── manifest.json        # Extension manifest (Manifest V2)
├── background.js        # Fetches release data, draws the icon, holds state
├── popup/
│   ├── popup.html       # Popup markup
│   └── popup.js         # Queries background state and renders the popup
└── icons/               # Toolbar icons (SVG)
```

## Development

Install dev dependencies (ESLint):

```bash
npm install
```

Lint the extension scripts:

```bash
npm run lint
```

Load the extension temporarily in Firefox:

1. Open `about:debugging`
2. Click **This Firefox**
3. Click **Load Temporary Add-on…**
4. Select `webextension/manifest.json`

## Building

A packaged `.zip` is generated by [web-ext](https://github.com/mozilla/web-ext):

```bash
cd webextension
web-ext lint
web-ext build
```

The output is placed in `webextension/web-ext-artifacts/`.
